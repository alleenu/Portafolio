Aplicación de Clima

En el desarrollo de la aplicación, utilicé la biblioteca React y Material UI para crear la interfaz de usuario. Además, realicé el deploy del sitio en Vercel para que esté disponible en línea.

Agencia de Viajes

Se trata de un sitio web de una agencia de viajes que ofrece información sobre diversos paquetes turísticos. Se ha implementado un formulario de cotización que calcula el costo del viaje en función de la cantidad de personas y las fechas seleccionadas. Tecnologías utilizadas HTML, CSS y JavaScript.

Tienda Online

Este proyecto es una implementación de una tienda en línea de ropa con funcionalidades de pago a través de PayPal. Está desarrollada utilizando la tecnología MERN Stack, que incluye MongoDB, Express, React y Node.js. La plataforma proporciona una experiencia de compra intuitiva, permitiendo la selección de productos.
